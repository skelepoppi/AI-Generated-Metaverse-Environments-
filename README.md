# PromptSpace: AI-Generated Metaverse Environments

**PromptSpace** is a decentralized platform that allows users to generate, own, and monetize high-fidelity 3D virtual environments using simple AI prompts.

PromptSpace bridges the gap between Generative AI and the Metaverse. By utilizing state-of-the-art 3D generative models and blockchain technology, it enables anyone to become a virtual world-builder without needing complex design skills.

## ‚ú® Key Features
- **Prompt-to-World Engine**: Generate 3D landscapes, interiors, and lighting setups from text or voice commands.
- **NFT Ownership**: Every generated environment is minted as a unique NFT on a low-fee Layer 2 blockchain (e.g., Polygon).
- **Dynamic AI Assets**: Real-time generation of 3D objects and furniture to populate your space.
- **Interoperability**: Export environments in standard `glTF` and `GLB` formats for use in existing Metaverse platforms like Decentraland or custom VR apps.
- **AI-Powered NPCs**: Integrate smart, interactive guides into your spaces using LLM-driven behaviors.
